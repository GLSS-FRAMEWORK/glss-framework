# Kill Chain Examples

Multi-segment attack scenarios demonstrating Ground → Link → Space kill chains.

## Example Scenarios

### Scenario 1: Cloud to Satellite
1. **Ground** — Cloud infrastructure compromise via supply chain attack
2. **Ground** — Lateral movement to mission control system
3. **Link** — Malicious command injection via uplink
4. **Space** — Firmware modification on satellite

### Scenario 2: GNSS Spoofing Chain
1. **Link** — GPS signal interception and analysis
2. **Link** — Spoofed signal generation and broadcast
3. **Space** — Satellite navigation data corruption
4. **Ground** — Downstream impact on ground-based positioning services
