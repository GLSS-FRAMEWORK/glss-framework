# Governance — GLSS Framework

## Fondation GLSS

Le GLSS Framework est gouverné par la **Fondation GLSS**, une association loi 1901 française, dont la mission est de promouvoir, développer et maintenir le framework.

## Structure de Gouvernance

### Membres Fondateurs

Les fondateurs disposent de droits de veto sur les changements fondamentaux du framework (modification de licence, changement d'architecture, dissolution).

- **Badaoui Salah** — Président
- **Hajer Boudhir** — Vice-Présidente
- **Achille Lele** — Secrétaire Général

### Rôles dans le Projet Open Source

#### Maintainers
- Droits de merge sur la branche `main`
- Validation des PRs majeures
- Décisions architecturales
- Actuellement : les 3 fondateurs

#### Committers
- Contributeurs réguliers ayant acquis la confiance de la communauté
- Peuvent merger les PRs mineures (corrections, documentation)
- Promus par consensus des maintainers

#### Contributors
- Toute personne soumettant des Pull Requests, Issues, ou documentation
- Pas de droits de merge directs

#### Advisory Board
- **Conseil Scientifique** (3-7 experts) : validation technique, orientation recherche
- **Conseil Consultatif Industriel** (membres industriels) : recommandations roadmap

## Processus de Décision

### Décisions techniques courantes
- Consensus entre maintainers
- En cas de désaccord : vote à majorité simple

### Décisions architecturales majeures
- Discussion publique via GitHub Issue/Discussion
- Période de commentaire de 14 jours
- Approbation par au moins 2/3 des maintainers

### Modifications fondamentales
- Changement de licence : unanimité des fondateurs + AGE
- Modification de l'architecture tri-segmentaire : unanimité des fondateurs
- Ajout/suppression de segment : unanimité des fondateurs + avis du Conseil Scientifique
