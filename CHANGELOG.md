# Changelog

All notable changes to the GLSS Framework will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.1.0] — 2026-02-15

### Added
- Initial public release of GLSS Framework
- Tri-segmentary architecture: Ground, Link, Space
- STIX 2.1 matrix format
- Mappings: ATT&CK, SPARTA, SPACE-SHIELD
- Compliance mappings: NIS2, CRA, NIST CSF v2.0
- 16 referentials articulated
- Project governance documentation
- Contributing guidelines
- Code of Conduct

### Framework Coverage
- Ground segment: Control centers, ground stations, IT/OT, cloud, MLOps, supply chain
- Link segment: RF uplink/downlink, optical links, ISL, TT&C protocols
- Space segment: Satellite bus, payload, firmware, OTA updates, embedded AI
